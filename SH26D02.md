# **Internship Project Proposal `SH26D02`**

## **Project Title: ERP Intelligent Notification & Alert Management System**

---

## **1. Problem Statement**
### **Client Context**
Our ERP clients currently face notification challenges:
- Receive generic, non-personalized system notifications
- Miss critical business alerts due to information overload
- Cannot customize alerts based on their specific business rules
- Have no self-service interface to manage notification preferences
- Struggle with timely responses to critical events (low stock, overdue invoices, etc.)
- **CRITICAL CONSTRAINT:** Customer ERP data is highly confidential and cannot be shared or exposed

### **Their Request**
> "We need a smart notification system where we can define our own business alerts. We want to choose which ERP module to monitor, set custom conditions, and receive timely alerts exactly when and how we need them."

---

## **2. Solution Overview**
We will build a **web-based notification management system** that:
- Provides a user-friendly interface for configuring custom alerts
- Integrates with ERP modules (Stock, CRM, Facturation, GMAO, GPAO, RH, etc.)
- Allows setting dynamic conditions (min stock, montant dépassé, date expiration, etc.)
- Sends notifications via email and potentially other channels
- Enables users to manage their notification preferences independently
- Runs both on-premise or cloud-based depending on client needs

## **6. Example Use Cases**

### **Stock Module Alerts:**
```
IF article.quantity < article.min_stock
AND article.category IN ["Electronics", "Perishables"]
THEN send email to "warehouse@company.com"
WITH priority = "HIGH"
AND repeat daily until resolved
```

### **Facturation Module Alerts:**
```
IF invoice.amount > 10000
OR invoice.due_date < TODAY + 3 days
THEN send in-app notification to manager
AND email to "accounting@company.com"
```

### **CRM Module Alerts:**
```
IF lead.status_changed_to = "Contacted"
AND lead.potential_value > 5000
AND lead.source = "Website"
THEN send Slack notification to sales_team
```

### **GMAO/GPAO Alerts:**
```
IF maintenance_task.due_date = TODAY
AND task.priority = "Critical"
THEN send SMS to technician
AND email to supervisor
```
